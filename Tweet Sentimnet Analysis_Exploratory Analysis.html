Loading a corpus or list corpus¶

Conc corpora are stored in a directory ending with .corpus. Locate the directory for the corpus you want to load. In this case, I want to load the machine-genereated essay corpus which is in a directory within save_path called machine-generated-essays.corpus. You can load it like this:

You don't have to use corpus for the variable name. It may be helpful to use a more descriptive name if you are working with multiple corpora (e.g. essay_corpus)

In [70]:
corpus = Corpus().load(f'{save_path}tweet-sentiment.corpus')

2.3 VADER Compound Score Distribution¶

Overlap between classes suggests ambiguity and explains why classification is difficult.

In [105]:
from vaderSentiment.vaderSentiment import SentimentIntensityAnalyzer
tweetana = SentimentIntensityAnalyzer()
def calcvader(text):
    scores = tweetana.polarity_scores(str(text))
    return pd.Series(scores)
vscores = df['text'].apply(calcvader)
vdf = pd.concat([df.copy(), vscores], axis=1)
